## 二、色彩转折点

### 转折点1：第8集8-1结束 → 8-2开始
- **从**：冷青+警告红（车内危机）
- **到**：暖金+大理石灰白（酒店奢华）
- **手法**：进酒店旋转门时，自然色温过渡
- **意义**：危机暂歇，进入下一战场

### 转折点2：第8集8-2结束 → 8-3开始
- **从**：暖金（酒店门口）
- **到**：金红+烛光琥珀（宴会厅深处）
- **手法**：林清月走入大门时，色彩自然加深
- **意义**：进入虎穴

### 转折点3 ⭐：镜23（周青摔杯）
- **从**：金红（暖金3200K）
- **到**：冷蓝杀机（5600K）
- **手法**：**色温在3秒内骤变**，配合钨丝灯故障般闪烁
- **意义**：**假面碎裂**——典型好莱坞"撕破假面"色彩信号
- **参考**：大卫·芬奇《消失的爱人》

### 转折点4 ⭐⭐：镜28→镜29（暴力升级 → 神迹降临）
- **从**：血红+阴影黑（兽性暴露）
- **到**：冷白神性+余烬橙（判官降临）
- **手法**：硬币飞过的瞬间，色彩**从血红极速过渡到银白**
- **意义**：**神性凌驾于兽性之上**

## 三、调色参数建议

### 8-1 车内戏（DaVinci Resolve 节点配置）

```
节点1（一级调色）：
- Lift（暗部）：Y=-5, R=-3, G=0, B=+5
- Gamma（中间调）：Y=0, R=0, G=-3, B=+8
- Gain（亮部）：Y=+5, R=-2, G=0, B=+3

节点2（二级调色）：
- Highlights：饱和度-15%
- Shadows：色温-300K（更冷）

节点3（特效层）：
- 警告红车尾灯：色相分离 H=355°, S=+30
```

### 8-3 宴会厅戏

```
节点1：
- Lift：Y=0, R=+3, G=-2, B=-5
- Gamma：Y=+3, R=+5, G=0, B=-5
- Gain：Y=-3, R=+8, G=+3, B=-8

节点2（暖金强化）：
- 整体色温+500K（更暖）
- 整体饱和度+10%
```

### 9-1 撕破脸（关键转折）

```
镜头前2秒：使用宴会厅参数
镜头后3秒：
- 色温骤降-1500K
- 整体降明度-10%
- 饱和度-20%（杀意冷感）
- 阴影加深至#0F1419
```

### 9-1 判官登场

```
镜头33A（剪影伫立）：
- 强烈逆光：背景5500K冷白
- 主体勾边光：3200K暖金
- 高对比度（增加150%）

镜头35（鸟瞰收束）：
- 中央吊灯3200K暖金（萧天位置）
- 四周5500K冷白（跪拜区）
- 形成"以萧天为中心的色温辐射圈"
```

## 五、AI生视频时的色彩控制

### 提示词中如何加入色彩

**示例（车内戏）**：
```
[角色描述], [动作], [场景], cinematic color grading, 
cool teal and orange shadows, color temperature 5000K, 
film noir lighting, ARRI Alexa look
```

**示例（撕破脸瞬间）**：
```
[角色描述], color temperature shift from warm 3200K 
to cool 5600K, dramatic color transition, dystopian 
mood, deep shadows, blue-orange contrast
```

**示例（判官登场）**：
```
[角色描述], silhouette against bright cool white 
backlight, warm amber rim light, cinematic religious 
icon composition, divine light effect
```
