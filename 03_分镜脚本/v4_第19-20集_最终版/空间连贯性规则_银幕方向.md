# 第19-20集 · 银幕方向与空间连贯性规则

> **用途**：确保分镜切换时人物位置不跳变，AI视频生成后拼接不穿帮
> **适用版本**：v7 AI视频生成版
> **核心原则**：180度轴线法则 + 固定银幕方向

---

## 一、轴线法则说明

两个角色对话/互动时，摄影机始终在他们连线的同一侧拍摄。
跨越轴线 = 人物在画面中的左右位置突然互换 = 观众空间感混乱。

---

## 二、第一场 · 早餐戏 · 轴线与银幕方向

### 座位固定（俯视）

```
        【画面上方 = 远端（远离门口）】

              落地窗（右墙）
                  │
    ┌─────────────┼──────────────┐
    │  凌云霄（远端短边，面朝镜头）  赵小清（右上暗角）│
    │             │              │
    │  苏婉清    餐桌     沈碧瑶  │
    │ （右长边）        （左长边） │
    │             │              │
    │     叶甜甜（近端短边，背朝镜头）│
    │             │              │
    └─────────────┼──────────────┘
                  │
             摄影机主位（门口）

        【画面下方 = 近端（靠近门口）】
```

**布局逻辑**：凌云霄在远端面朝镜头，其余三女视线自然朝他 = 朝镜头方向，主镜头一次拍到所有人的脸。叶甜甜在近端主镜头里是背影，但她的特写从远端反打即可。

### 银幕方向固定规则

| 角色 | 在画面中的固定位置 | 视线方向 |
|------|------------------|---------|
| 凌云霄 | 画面中央偏上（远端） | 向下（面朝镜头/朝门口） |
| 沈碧瑶 | 画面左侧 | 向右上（朝凌云霄） |
| 苏婉清 | 画面右侧 | 向左上（朝凌云霄） |
| 叶甜甜 | 画面中央偏下（近端） | 向上（朝凌云霄，主镜头背影） |
| 赵小清 | 画面右上角（远端暗角） | 向左下（朝凌云霄） |

### 轴线设定

- **主轴线**：凌云霄↔叶甜甜（纵向），摄影机在门口端（画面下方）向远端拍
- **副轴线1**：凌云霄↔沈碧瑶（斜向左下），拍沈碧瑶时她在画面左，视线朝右上
- **副轴线2**：凌云霄↔苏婉清（斜向右下），拍苏婉清时她在画面右，视线朝左上
- **间谍轴线**：凌云霄↔赵小清（纵深同侧），赵小清始终在画面右上暗角，偷瞄方向朝左下

### 每镜位置标注规则

格式：`【银幕位置】角色在画面 X 侧，视线朝 Y 方向`

## 三、第二场 · 车内戏 · 轴线与银幕方向

### 座位固定（俯视）

```
    ┌──────────────────┐
    │    挡风玻璃（上）  │
    │                  │
    │  凌云霄    沈碧瑶  │
    │ （画面左） （画面右）│
    │                  │
    └──────────────────┘
```

### 银幕方向固定规则

| 拍摄方式 | 凌云霄位置 | 沈碧瑶位置 |
|---------|----------|----------|
| 从前方拍入（透过挡风玻璃） | 画面左 | 画面右 |
| 拍凌云霄特写（从副驾方向看） | 画面左，面朝右 | 不在画面 |
| 拍沈碧瑶特写（从驾驶位方向看） | 不在画面 | 画面右，面朝左 |
| 从后排向前拍 | 画面左 | 画面右 |
| 后视镜画面 | 镜像翻转（凌云霄右，沈碧瑶左） | — |

### 轴线设定

- **主轴线**：凌云霄↔沈碧瑶（横向），摄影机始终在轴线前方（挡风玻璃侧）或后方（后排侧）
- **绝不跨越**：不从车顶或车底拍（会导致左右互换）
- **壁咚戏例外**：镜26从副驾背后斜45度拍，此时沈碧瑶背部在画面中央，凌云霄从画面左侧倾入——这是唯一允许的"半跨轴"，因为壁咚本身就是空间入侵

## 四、第二十集 · 追逐戏 · 轴线与银幕方向

### 运动方向固定

| 车辆 | 运动方向 | 银幕方向 |
|------|---------|---------|
| 凌云霄的白色SUV | 始终从画面左→右（或下→上） | 画面前方/右侧 |
| 周锋的黑色车队 | 始终从画面右→左追赶（或上→下） | 画面后方/左侧 |

### 关键规则

1. **追逐方向不变**：白车永远朝画面右/上方跑，黑车永远从画面左/下方追。航拍时白车在前黑车在后。
2. **周锋车内**：周锋坐副驾（画面右），司机在画面左。视线朝画面右前方（追赶方向）。
3. **江霞路断口**：白车从画面左冲向画面右的断口。断口在画面右侧。对岸在更右侧（画面外）。
4. **轰炸机方向**：从画面右侧飞入（与白车同向），尾舱坡道朝画面左侧放下（接住从左冲来的白车）。
5. **落地后**：白车在断口右侧（对岸），周锋车队停在断口左侧。中指朝画面左侧竖起。

## 五、切镜连贯性检查清单

每次切镜时必须确认：

- [ ] 同一角色在前后两镜中的画面位置是否一致？
- [ ] 视线方向是否连贯？（A看向画面右 → 下一镜B在画面右看向左）
- [ ] 运动方向是否连贯？（车向右开 → 下一镜车仍向右）
- [ ] 是否跨越了轴线？如果跨越，是否有过渡镜（如正面镜/俯视镜）？
- [ ] 后视镜画面是否做了镜像翻转标注？
